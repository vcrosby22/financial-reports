# Financial Reports

Daily market analysis report — auto-updated on weekdays via GitHub Actions.

**[View the latest report →](https://vcrosby22.github.io/financial-reports/)**

## What's in the report

- **Market Overview** — prices, returns, RSI, technical signals for 80+ assets (indices, 50+ stocks, sector ETFs, crypto, forex)
- **Macroeconomic Indicators** — FRED data: yield curve, credit spreads, unemployment, consumer confidence, Fed funds rate
- **Fundamentals** — earnings revisions, insider activity, analyst targets, financial health for all watched stocks
- **Opportunities** — long and short ideas with risk-adjusted scoring, thesis, risks, and position sizing
- **Risk Signals** — 3-layer assessment (technical + macro + fundamental) with a 0–100 score, confidence, and methodology in the collapsible legend

## Update schedule

Reports run **twice on weekdays** (US Eastern Time):

- **~7:00 AM ET** — before the market open (overnight / prior close data)
- **~4:30 PM ET** — after the market close (full session data)

You can also trigger a manual run from the **Actions** tab.

## Disclaimer

This is analysis for educational purposes, not financial advice.
