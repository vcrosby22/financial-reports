# Financial Reports

Daily market analysis report — auto-updated on weekdays via GitHub Actions.

**[View the latest report →](https://vcrosby22.github.io/financial-reports/)**

## What's in the report

- **Market Overview** — prices, returns, RSI, technical signals for 55+ assets (stocks, ETFs, crypto, forex)
- **Macroeconomic Indicators** — FRED data: yield curve, credit spreads, unemployment, consumer confidence, Fed funds rate
- **Fundamentals** — earnings revisions, insider activity, analyst targets, financial health for 26 stocks
- **Opportunities** — long and short ideas with risk-adjusted scoring, thesis, risks, and position sizing
- **Risk Signals** — 3-layer assessment (technical + macro + fundamental) with confidence scoring

## Update schedule

Reports are generated automatically at **7:00 AM ET on weekdays** (before market open). You can also trigger a manual update from the Actions tab.

## Disclaimer

This is AI-assisted analysis for educational purposes, not financial advice.
